# Seminari_flutter_backend

This repository contains the backend for the **Flutter Seminar** project.  
It serves as the server-side component that supports the Flutter application.

This backend is an **improved version of the JWT Seminar**, expanding its functionality to integrate with the Flutter frontend and provide secure communication between client and server.

## Overview

- Built to complement the Flutter Seminar project.  
- Implements authentication and data handling.  
- Based on concepts learned in the **JWT Seminar** and enhanced for real use.
